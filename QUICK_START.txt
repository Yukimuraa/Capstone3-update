❌ TROUBLESHOOTING:

   Problem: "Connection failed"
   Solution: 
   • Check XAMPP MySQL is running (green in control panel)
   • Check config/database.php:
     - Database: chmsu_bao
     - User: root
     - Password: (empty)

   Problem: "Page not found"
   Solution:
   • Make sure Apache is running in XAMPP
   • Check URL: http://localhost/Capstone-3/
   • Check folder is in C:\xampp\htdocs\

   Problem: "Setup completed but still error"
   Solution:
   • Run test_database.php to verify
   • Check all tables show ✓
   • Clear browser cache
   • Try different browser

   Problem: "Access denied"
   Solution:
   • Run XAMPP as Administrator
   • Check MySQL user permissions in phpMyAdmin
