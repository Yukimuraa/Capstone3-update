AUTOMATIC DETECTION:
   Any item name containing these keywords will show sizing:
   
   ✓ shirt      ✓ pants      ✓ jacket
   ✓ t-shirt    ✓ shorts     ✓ hoodie
   ✓ tshirt     ✓ polo       ✓ jersey

EXAMPLES:
   ✅ "BSIT OJT - Shirt"    → Shows sizing ✓
   ✅ "P.E T-Shirt"         → Shows sizing ✓
   ✅ "Basketball Pants"    → Shows sizing ✓
   ✅ "Department Polo"     → Shows sizing ✓
   ❌ "ID Cord"             → No sizing
   ❌ "Cap"                 → No sizing

AVAILABLE SIZES:
   □ XS (Extra Small)
   □ S (Small)
   □ M (Medium)
   □ L (Large)
   □ XL (Extra Large)
   □ 2XL
   □ 3XL
