    SHOPPING CART SYSTEM - QUICK START GUIDE
═══════════════════════════════════════════════════════════════

✅ STEP 1: RUN DATABASE SETUP
────────────────────────────────────────────────────────────────
Open your browser and navigate to:

   http://localhost/Capstone-3/database/setup_cart_system.php

This will automatically:
   • Create the cart table
   • Add batch_id column to orders table
   • Add size column to orders table
   • Create necessary indexes

✅ STEP 2: TEST STUDENT CART FEATURE
────────────────────────────────────────────────────────────────
1. Login as a student
2. Go to "Available Items"
3. Click "Add to Cart" on any item
4. Click "Shopping Cart" or "View Cart" button
5. Add more items or proceed to checkout
6. Place your order
7. View the batch receipt

✅ STEP 3: TEST STAFF CART FEATURE
────────────────────────────────────────────────────────────────
1. Login as staff
2. Go to "Inventory Management"
3. Click "Order Items" tab at the top
4. Click "Add to Cart" on any item
5. Click "View Cart" button
6. Complete checkout process

    TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

Problem: Cart badge not updating
Solution: Check JavaScript console, verify add_to_cart.php works

Problem: Items disappear from cart
Solution: Check cart table exists, verify user login

Problem: Can't add items
Solution: Run setup_cart_system.php, check inventory stock

Problem: Checkout fails
Solution: Check orders table has batch_id column
