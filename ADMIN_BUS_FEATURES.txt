┌───────────────────────────────────────────────────────────────────────────┐
│                       WORKFLOW EXAMPLES                                   │
└───────────────────────────────────────────────────────────────────────────┘

    📌 WEEKLY ROUTINE:
    Monday:    Check pending requests → Approve/Reject
    Tuesday:   Check bus status → Update if maintenance needed
    Wednesday: Monitor fuel prices
    Thursday:  Update fuel rate if price changed significantly
    Friday:    Generate weekly reports

    🔧 MAINTENANCE WORKFLOW:
    1. Bus needs service
       → Go to Manage Buses
       → Click Edit on the bus
       → Change status to "Maintenance"
       → Click Update

    2. Maintenance complete
       → Click Edit on the bus
       → Change status to "Available"
       → Bus can now be booked again!

    ⛽ FUEL RATE UPDATE WORKFLOW:
    1. Check local gas stations
    2. Note the average price
    3. Go to Fuel Rate tab
    4. Update the rate
    5. Inform staff about the change
    6. New bookings use new rate automatically!

┌───────────────────────────────────────────────────────────────────────────┐
│                         COLOR CODING                                      │
└───────────────────────────────────────────────────────────────────────────┘

    Bus Status Colors:
    🟢 GREEN   = Available (ready to book)
    🟡 YELLOW  = Booked (currently in use)
    🟠 ORANGE  = Maintenance (being serviced)
    🔴 RED     = Out of Service (not available)

    Request Status Colors:
    🟡 YELLOW  = Pending (awaiting approval)
    🟢 GREEN   = Approved (confirmed)
    🔴 RED     = Rejected (denied)
    ⚫ GRAY    = Cancelled (user cancelled)
